# Vybe Coding 101 — Course Plan

## overview

**Title:** Vybe Coding 101
**Tagline:** *“Learn to build, not just code.”*
**Tools:** V0, Cursor, CLI, Github
**Target Audience:** Absolute beginers with no coding experince who want to learn how to vybe code
**Vibe:** Creative, visual, hacker energy — build 7 projects in days 7.
**Goal:** Turn total beginners into confident creators who can ship small, functional web projects on their own.

## learning objectives

- teach how to build landing pages using v0 & vercel
- teach how internet works - servers and client 
- teach what CLI is and how it works 
- teach git and github 
- teach API 
- teach what databases are and how they works 
- teach what authentication is and how it works
